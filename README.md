# Arte Buffet - Site Institucional

Site institucional do Arte Buffet, especializado em eventos de luxo em Fortaleza.

## Tecnologias Utilizadas

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- React Router

## Como executar o projeto

### Pré-requisitos

- Node.js & npm instalados - [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Passos para execução

```sh
# Passo 1: Clone o repositório
git clone <URL_DO_REPOSITORIO>

# Passo 2: Navegue até o diretório do projeto
cd artebuffetconceito

# Passo 3: Instale as dependências
npm install

# Passo 4: Inicie o servidor de desenvolvimento
npm run dev
```

O projeto estará disponível em `http://localhost:8080`

## Scripts Disponíveis

- `npm run dev` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria a build de produção
- `npm run preview` - Visualiza a build de produção
- `npm run lint` - Executa o linter

## Estrutura do Projeto

- `src/pages/` - Páginas da aplicação
- `src/components/` - Componentes reutilizáveis
- `src/components/ui/` - Componentes de UI (shadcn-ui)
- `src/lib/` - Utilitários e helpers
- `src/hooks/` - Custom hooks
